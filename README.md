## TP 2

### Description de l'application
- L'utilisateur peut cliquer sur un bouton « + » qui le redirige vers un nouvel écran « Ajouter une carte ».
- L'écran « Ajouter une carte » comporte un bouton Annuler permettant de revenir à l'écran principal.
- L'utilisateur peut saisir et soumettre une question et une réponse via l'écran « Ajouter une carte ».
-  L'utilisateur verra la carte qu'il vient de créer sur l'écran principal lorsqu'il appuiera sur le bouton « Enregistrer ».
- Partagez votre progression sur GitHub !
